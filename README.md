# Ecommerce Selenium Automation Framework

## Project Overview
This project is a Selenium-based test automation framework developed using Java, TestNG, and Maven.
It automates core functionalities of an e-commerce web application.

## Tech Stack
- Java 17
- Selenium WebDriver
- TestNG
- Maven
- GitHub

## Features Automated
- Login functionality (valid and invalid scenarios)
- Product listing validation
- Add product to cart
- Cart navigation

## Framework Design
- Page Object Model (POM)
- Base Test setup with reusable WebDriver initialization
- TestNG annotations for test execution

## Execution
Tests can be executed using:
- Maven command: `mvn test`
- TestNG runner

## Status
- Core automation completed
- Enhancements in progress (Reports, Data-driven testing)
